## 2. サーバー独自のガイドライン

Marshal Serviceでは、ユーザーが安全かつ円滑にサービスを利用できるよう、以下の独自ルールを設けています：

- 各チャンネルや機能ごとに**専用の利用規約・ガイドライン**があります。必ず事前に確認してください。
- 当サーバーでは、**限定保証制度**を提供しています。対象カテゴリーごとのガイドラインを熟読のうえご利用ください。
- 過度にスタッフの注意を引く行動（執拗なPing、不適切な発言など）はお控えください。
- チケット制度を導入していますが、**不必要なチケット連投やサポートへの迷惑行為**は処分対象です。
- 本サーバーは**専属Developerの監修下で運営**されています。
  - Staff・Developerへの勧誘、買収、外部持ち出し等は**重大な規約違反**と見なされます。
  - 自動販売機・各種BotはMarshal Service専用であり、**外部への販売・導入サポート等は一切行っておりません。**

## 3. 禁止事項（特に注意）

- **コミュニティエリアでの商品クレーム投稿は禁止**です。問題がある場合はチケットサポートをご利用ください。
- 他のサーバーで購入した商品に関する議論・クレームも**当サーバーでは一切禁止**とします。Marshal Serviceと無関係の話題で混乱を招く行為は厳しく対処されます。

## 4. 違反への対応について

- 軽微な違反に対しては、まず**警告・注意喚起**を行います。
- 悪質または重大な違反に関しては、**警告・通告なしでBAN処分**とする場合があります。
- メンバー間のトラブル・詐欺等に関しては、**当サーバーは一切の責任を負いません**。
