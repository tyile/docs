## サービス概要（About）

Marshal Serviceは、**Discord上で展開されるアカウント自動販売サービス**です。\
当サービスが提供する販売ボットを通じて、ユーザーはPayPayリンクを使って商品を即購入でき、**支払い完了後は自動でアカウント情報が送信されます**。

## 取り扱い商品（Available Products）

Marshal Serviceでは、以下のようなアカウントを幅広く取り扱っています：

- **モンストの石垢・進行垢（メインカテゴリ）**\
  業界最大級の在庫数を誇り、豊富な属性・初期キャラの組み合わせをご用意。
- **Valorant（PCゲームカテゴリの中心）**\
  正規引退アカウントから、安価なスマーフ向けNFA（電話番号未認証）アカウントまで、幅広く取り揃えています。
- **その他人気タイトル**\
  原神、Apex Legends、Fortnite、ブルアカ、FGO など、スマホ・PC問わず多数の人気ゲームアカウントを取り扱い中。

## 取引の流れ（Transaction Flow）

1. ユーザーが希望のアカウントを選択
2. Discord上の自動販売機に、PayPayリンクを送金
3. 決済確認後、自動でアカウント情報がDMに送信される
4. 取引完了（連絡不要・即時受け取り）
